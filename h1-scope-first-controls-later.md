# ISMS-soveltamisala  
## a) Perustaso  
1. Soveltamisalaani kuuluu reititin, Wi-Fi, Windows 11 pöytäkone, Windows 11 kannettava tietokone, Android puhelin, Linux Debian VM, kurssimateriaalit, tunnukset tarvittaviin sovelluksiin.
2. Rajaan ulos Playstation 5, älylampun, aly-TV:n ja macbookin. Nämä siksi, että ne ovat merkityksettömiä kurssin näkökulmasta.
3. Keskeisiä rajapintoja ovat Github, johon teen tehtävät, Haaga-Helian Moodle, Laksu, jonne tehtävät palautetaan, koulun OneDrive, reititin (ZTE), operaattori (DNA)
